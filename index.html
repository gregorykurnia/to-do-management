Connect Google Calendar
One-time setup (5 minutes)
  1. Go to console.cloud.google.com → create or select a project
  2. APIs & Services → Enable APIs → search Google Calendar API → Enable
  3. APIs & Services → Credentials → Create Credentials → OAuth client ID
  4. Application type: Web application
  5. Authorized JavaScript origins: add
  6. Copy the Client ID and paste it below

After connecting, you'll see a Google sign-in popup. Access tokens refresh automatically (valid 1 hour; click Connect again to refresh if needed).
